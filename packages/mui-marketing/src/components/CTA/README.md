## CTA

Call to action section

---

#### Examples

**Simple**

```jsx { "props": { }}
const body = `Lorem ipsum dolor sit amet consectetur adipiscing elit..`;
const actions = [
  {
    url: '#1',
    content: 'Buy now',
  },
];
<CTA whitespace center title="My Section" body={body} actions={actions}>
  <p>Some page content</p>
</CTA>;
```
